Is openlava the same things as Platform Lava?

openlava is based on Platform Lava but there are many differences.

Do I have to pay to use openlava?

No there's no charge to use openlava and there never will be.

How can I get technical support?

There are several ways you can get overlava technical support

  • You can join the openlava-discussion mailing list.
  • Read the wiki on github
  • Read the messages on the forumn or post a message and ask the openlava community (not yet implemented)

How can I become an openlava developer?

The openlava source code is hosted on github.com. You can download it, fork a branch, and start developing.

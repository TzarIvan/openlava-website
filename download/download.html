OpenLava downloads

OpenLava is distributed as source code tarball or it can be directly cloned from the github repository.

OpenLava Source Code

OpenLava 3.0b beta version is available on github. It is the master branch with 3.0b tag. The 3.0 branch will be cut when 3.0 will be officially released in April 2015. The major features are hierarchical fairshare and preemption. OpenLava has also an integration with docker containers. 3.0b rpms are to download from Teraproc . The tarball can be downloaded from

OpenLava 3.0 tarball

OpenLava 2.2 is the most stable version of the code. OpenLava can be compiled and installed using 'make install'. OpenLava RPM is available for download from Teraproc . For a quick start using OpenLava check the documentation tab.

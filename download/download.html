openlava downloads

openlava is distributed as source code tarball or it can be directly cloned from the github repository.

openlava Source Code

openlava 2.2 is the most recent version of the code. openlava can be compiled and installed using 'make install'. openlava RPM is available for download from Teraproc . For a quick start using openlava check the documentation tab.

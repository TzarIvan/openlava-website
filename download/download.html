OpenLava downloads

OpenLava is distributed as source code tarball, it can be directly cloned from the github repository or downloaded as rpm file.

OpenLava Source Code

OpenLava 3.0b beta version is available on github. It is the master branch with 3.0b tag. The 3.0 branch will be cut when 3.0 will be officially released in April 2015. The major features are hierarchical fairshare and preemption. OpenLava has also an integration with docker containers.

The tarball can be downloaded from:

  • OpenLava 3.0 tarball

  • OpenLava 2.2 is the most stable version of the code.

    OpenLava 2.2 tarball

The source code tree is available from a git repository:

  • git clone git://github.com/openlava/openlava.git

OpenLava RPM

OpenLava RPM for both 2.2 and 3.0b are available for download from Teraproc .
